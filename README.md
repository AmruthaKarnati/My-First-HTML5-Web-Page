# My-First-HTML5-Web-Page
Beginner HTML5 practice project: My First HTML5 Project: Comprehensive beginner tutorial site showcasing HTML5 semantics, tables (daily routine), optimized images (Caribbean/Cancun vacations), advanced forms (validation, selects, checkboxes), and navigation.

**This project is based on HTML** Basic HTML5 project created to practice core concepts 

## Topics Covered
- HTML5 structure & semantics (`<!DOCTYPE>`, `<meta>`, `<article>`, `<nav>`)
- Headings (`<h1>`-`<h3>`), paragraphs, lists (`<ul>`, `<ol>`, `<dl>`)
- Attributes (id, href, src, alt, datetime, loading="lazy/eager", required, pattern)
- Images (`<figure>`, `<figcaption>`, optimization)
- Tables (`<table>`, thead/tbody/tfoot, rowspan/colspan, `<time>`)
- Forms (text/password/tel/number/select/checkbox/radio/textarea, labels, validation)

## Project Description
Simple webpage titled "My First Web Page" showcasing **pure HTML5** learning: personal about section, HTML5 logo/code sample, daily schedule table ("eat, sleep, code"), vacation images (Caribbean beach eager-loaded, Cancun lazy-loaded), places to avoid (Poles), and interactive contact form (demo to httpbin.org—**fix `messgae`→`message`**).
